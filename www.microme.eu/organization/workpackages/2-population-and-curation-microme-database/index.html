2 - Population and curation of Microme database

Workpackage No.: 
2
Workpackage Title: 
Population and curation of Microme database
Activity Type: 
RTD
Objectives: 
<p> Provide a high quality database of microbial metabolic pathways.</p>
Description: 
<p> This work package aims at populating and enriching Microme database with reference pathways and species specific pathway assemblies (SSPAs). This work will proceed in two phases; an initial bootstrapping phase, and an ongoing process thereafter. Automatic data import (in the first phase) and generation (in the second phase, through a procedure for pathway projection implemented in work package one) will be intimately linked to curation throughout the lifetime of the project. Task 1 Specification for Microme curation and pathway projection Specifications: We will commence work by preparing a key specification document (D2.1) that will guide WP2-related software development performed in WP1. This document will contain two parts: (i) the specification of the Microme data curation process related to reference pathway curation and functional annotation (GRP families assigned to genes/proteins) in order to guide the development of Microme curation environment (D1.3) (ii) the specification of the pathway projection strategy in order to guide the development of Software for pathway projection (D1.4, D1.8, D1.9). After the first Microme annotation jamboree is held (see task 2), the experience gained will be used to finalize a curator-guide that will specify standard operating procedures (SOPs) followed by curators to create or update Microme data (D2.4) during its mature phase Task 2 Preparation of data set for initial public release Pre-release data set In order to enable the launch of the Microme project, we will bootstrap the database through the automatic input of data from publicly available resources, for genomes (e.g. Genome Reviews, PkGDB), enzymes (e.g. UniProtKB, IntEnz), metabolites (e.g. ChEBI) and pathways (e.g. UniPathway, BioPath, Reactome, KEGG, EcoCyc) (D2.2). A number of the Microme beneficiaries are actively involved in maintaining several of these resources. Redundancies between imports will be identified; overlaps or inconsistencies signalled for manual examination; and automatic validation techniques applied. Valid data will be fitted to the Microme data model. To facilitate this process, the data warehousing system OBIwarehouse (http://www.grenoble.prabi.fr/obiwarehouse), to whose development SIB has contributed, will be used. OBIwarehouse has already been used successfully to integrate some of the relevant data and will be extended to cover the additional data sources relevant to Microme. Initial Curation Priorities: Microme has a clearly defined set of priorities for its intitial phase, related to the bootstrapping of the resource, the expertise possessed by Consortium members, and the need to lay a foundation for certain activities planned for the second half of the project. The goal of the initial phase is to enable the launch of the Microme web portal; and provide readily-usable input data to the stoichiometric model reconstruction pipeline, described in WP3 and for the comparison of metabolic spaces through constrained-based models described in WP4. After the initial release, Microme will be guided by the community on subsequent curation priorities, according to the advice of its Microbiology Committee. Our initial priority will be given to reconstructing the species-specific pathways assemblies of E.coli (a reference microorganism) and three other reference organisms on which Microme beneficiaries have in-depth scientific expertise: Bacillus subtilis 168 (a paradigm of industrially relevant bacterium), Acinetobacter baylyi (an bacterial species highly competent for natural transformation and hence for the construction of novel metabolic systems), and Pseudomonas aeruginosa (a threatening, opportunistic pathogen). This first set of high-quality metabolic networks will cover a sizeable fraction of reference pathways from the core metabolism (B.subtilis, P.aeruginosa and E.coli), but will also initiate our exploration of niche-specific metabolism (A.baylyi). Microme first public release: In addition to the pre-release data, the database prepared for the first public release of Microme (D2.3), will contain: (i) 4 curated SSPAs (Escherichia coli, Bacillus subtilis, Acinetobacter baylyi and Pseudomonas aeruginosa). (ii) Projection of curated reference pathways for amino acid biosynthesis to five hundred bacterial genomes. To achieve the Microme first public release all beneficiaries will be involved in jamboree (arranged as part of WP7) to review the bootstrapped data and extend the resource to ensure comprehensive coverage of the metabolic pathways of four pilot species. This will provide input for initiating stoichiometric modeling in work package 3 and for performing the comparative and evolutionary analyses described in work package 4. Task 3 Preparation of data sets for post-launch releases Production phase: The enrichment of the Microme database will continue throughout the lifespan of the project through human curation, and improved predictions through the software developed in work package 1. In addition, this curation process will benefit from feedback from the stoichiometric modelling and the comparative pathway analysis provided by work package 3 and 4. Microme SSPAs curation will be performed by the Genoscope, Sanger and SIB Microme staff. Reference pathways and Gene-Reaction-Protein families curation will be performed by the SIB Microme staff supported by the UniProt consortium curators involved in microbial annotation. In addition, we aim to recruit microbial experts from the community to assist in the curation process (in a process described in work package 7). During the production phase, work will focus on the initially prioritized Microme species (Escherichia coli, Salmonella and Pseudomonas genus), the set of genomes studied by expert genome annotators (at least 10 different species including mostly plant symbiotic bacteria and other environmental bacteria able to metabolize various metal compounds), and a set of additional metabolic pathways prioritised according the requirements of work package 5. There will be three major releases during the production phase: Microme data set release 2.0 (D2.5) will contain 6 curated SSPAs additionally to the automatically projected SSPAs for all bacterial genomes; and 2,000 biochemical reactions structured in 400 reference pathways. Microme data set release 3.0 (D2.6) will contain 10 curated SSPAs additionally to the automatically projected SSPAs for all bacterial genomes; and 3,000+ biochemical reactions structured in 600 reference pathways. Microme data set release 4.0 (D2.7) will contain 20+ curated SSPAs additionally to the automatically projected SSPAs for all bacterial genomes; and 5&rsquo;000+ biochemical reactions structured in 800 reference pathways.</p>
